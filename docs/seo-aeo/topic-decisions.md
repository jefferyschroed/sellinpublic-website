## Decision Rules

- Use `create_or_refresh_packet` only when the topic can support a source-backed standalone page.
- Use `resolve_gap_first` when the idea is strategically useful but evidence, examples, SME notes, or POV is not ready.
- Use `map_as_h2_faq_or_section` when the answer helps AEO clarity but does not deserve its own URL.
- Use `park_merge_or_retire` when the topic is generic, weakly related, or likely to create duplicate content.
